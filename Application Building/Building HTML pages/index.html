PROJECT DESCRIPTION


Vehicle damage detection is used to reduce claims leakage during insurance processing.
Vehicle inspection and validation are usually done. As it takes a long time, because a person needs to come and inspect the damage.
Here we are trying to automate the procedure. Using this automation, we can avoid time consumption for insurance claim procedure.
